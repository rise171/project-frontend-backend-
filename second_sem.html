Обзор курса

Во втором семестре мы углубимся в бэкенд-разработку, изучая различные технологии и подходы к созданию серверной логики, работы с базами данных и API.

Модули курса

Модуль 1

Основы серверного программирования

Модуль 2

Работа с базами данных (SQL и NoSQL)

Модуль 3

Создание RESTful API

Проекты

В рамках курса студенты будут работать над следующими проектами:

Название проекта Описание Срок выполнения
Приложение для заметок Создание сервиса для хранения и управления заметками. 4 недели
Система управления задачами Разработка API для управления задачами в команде. 6 недель

Ресурсы

Рекомендуемые материалы для изучения:

Node.js

Популярная платформа для разработки серверных приложений на JavaScript.

MongoDB

Документо-ориентированная NoSQL база данных для хранения данных.

RESTful API

Архитектурный стиль для создания веб-сервисов, взаимодействующих через HTTP.

Приложение для заметок

Создание сервиса для хранения и управления заметками. Студенты научатся работать с CRUD-операциями и базами данных.

Система управления задачами

Разработка API для управления задачами в команде. Студенты будут изучать аутентификацию и авторизацию пользователей.
